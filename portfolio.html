Snake Game

In this project the classical Snake game is recreated. The game has a 2 dimensional graphical user interface. In this game the snake can be controlled and directed to all 4 directions. Objects(food) will appear over the screen at random positions. The sanke has to directed over those objects(eat) to gather points. The points are also recorded in a scoreboard. As the score increases the difficulty level increases by increasing the speed of the snake's movement. The game gets over when the snake collides with itself.

Language used : Python
labraries used: turtle , random , time.
Github link : https://github.com/Smara2026/snakeGame
